# 📊 Health Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing a real-world dataset related to **Nutrition, Physical Activity, and Obesity**.
The goal is to extract meaningful insights using **Exploratory Data Analysis (EDA), Visualization, and Statistical Techniques**.

---

## 🎯 Objectives

* Analyze distribution of health data using histograms and boxplots
* Identify relationships between variables using scatter plots
* Find top contributing locations using bar and pie charts
* Study trends over time using line charts
* Perform correlation analysis between numerical features
* Apply hypothesis testing to validate results

---

## 📁 Dataset

* **Name:** Nutrition, Physical Activity, and Obesity Dataset
* **Type:** Mixed (Numerical + Categorical)
* **Key Columns Used:**

  * `Data_Value` → Health indicator value
  * `Sample_Size` → Number of samples
  * `YearStart` → Year of data
  * `LocationDesc` → Location/Region

---

## 🔍 EDA (Exploratory Data Analysis)

* Checked dataset shape and structure
* Identified missing values
* Performed statistical summary
* Cleaned data by handling missing values

---

## 📊 Visualizations Used

* Histogram → Distribution analysis
* Boxplot → Outlier detection
* Count Plot → Records per year
* Scatter Plot → Relationship between variables
* Bar Chart → Top locations
* Pie Chart → Proportion of locations
* Heatmap → Correlation analysis
* Line Chart → Trend over years

---

## ⚙️ Techniques Applied

* **IQR Method** → Outlier detection
* **Correlation Analysis** → Relationship between variables
* **Hypothesis Testing (T-Test)** → Statistical validation

---

## 🧠 Key Insights

* Data distribution shows variability across locations
* Certain regions contribute more to the dataset
* Moderate relationship observed between variables
* Trends over years indicate variation in health indicators

---

This project demonstrates how data analysis techniques can be used to understand real-world health datasets.
